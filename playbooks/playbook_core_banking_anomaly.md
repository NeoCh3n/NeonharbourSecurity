# Playbook: Core Banking Transaction Anomaly

- Validate anomaly with operations team (HK time zone coverage).
- Review Splunk transaction logs and Sentinel correlation rules.
- Cross-check with fraud analytics and MAS/HKMA reporting thresholds.
- If malicious, switch to disaster recovery instance with HKMA notification.
- Record MTTA, MTTI, MTTR for KPI dashboard updates.
